## ðŸ¤– AI ESTIMATOR SPECIFICATIONS

### **Phase 1: Basic Calculator**
```typescript
// Basic estimator structure for MVP
interface EstimatorInput {
  projectType: 'commercial' | 'medical' | 'religious' | 'industrial' | 'winery';
  squareFootage: number;
  stories: 1 | 2 | 3;
  timeline: 'standard' | 'premium' | 'expedite';
  location: 'tri-cities' | 'spokane' | 'other';
}

interface EstimatorOutput {
  costRange: {
    min: number;
    max: number;
    confidence: number; // percentage
  };
  timeline: {
    phases: TimelinePhase[];
    totalWeeks: number;
    startDate: Date;
    completionDate: Date;
  };
  nextSteps: string[];
}
```

### **Timeline Phases (Standard Speed)**
1. **Foundation/Concrete**: 2-3 weeks
2. **Framing**: 3-4 weeks
3. **Exterior Walls**: 2-3 weeks
4. **Roofing**: 1-2 weeks
5. **Systems (MEP)**: 4-6 weeks
6. **Drywall & Paint**: 3-4 weeks
7. **Cabinets**: 2-3 weeks
8. **Flooring**: 1-2 weeks
9. **Trim & Finishes**: 2-3 weeks
10. **Landscaping**: 1-2 weeks

### **Speed Multipliers**
- **Standard**: 1.0x (base timeline)
- **Premium**: 1.15x (+15% timeline, higher quality)
- **Expedite**: 1.30x (+30% timeline, rush job)

### **Seasonal Restrictions**
- **Concrete, Landscaping, Roadways**: Restricted Oct 15 - Mar 31
- **Timeline Adjustment**: Add 2-4 weeks if project includes restricted work during winter
